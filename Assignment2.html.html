  4. Create an ordered list of HTML tags. Each list item must include the tag name and some information about the tag.

    Ans= 1. `<html>`:

    • Defines the root element of an HTML document.

    • Contains all other HTML elements on the page.


    1. `<head>`:

      • Contains meta-information about the HTML document.

      • Typically includes the `<title>`, `<meta>`, `<link>`, and `<script>` tags.


    2. `<title>`:

      • Sets the title of the HTML document, displayed in the browser's title bar or tab.

    3. `<meta>`:

      • Provides metadata about the HTML document, such as character encoding and viewport settings.


    4. `<link>`:

      • Specifies external resources like stylesheets (CSS) or icons.

      • Used to link to external files.


    5. `<style>`:

      • Defines inline CSS styles or includes CSS code within the document.


    6. `<script>`:

      • Embeds JavaScript code in the HTML document or links to an external JavaScript file.


    7. `<body>`:

      • Contains the visible content of the web page, including text, images, and other media.


    8. `<h1>`, `<h2>`, `<h3>`, `<h4>`, `<h5>`, `<h6>`:

      • Headings of various levels, with `<h1>` being the highest (most important) and `<h6>` the lowest.

      • Used to structure and format text content.


    9. `<p>`:

      • Represents a paragraph of text.


    10. `<a>`:

      • Defines a hyperlink.

      • Used to link to other web pages or resources.

    11. `<img>`:

      • Embeds an image on the web page.

      • Requires a `src` attribute to specify the image file.


    12. `<ul>`:

      • Creates an unordered (bulleted) list.

      • Contains one or more `<li>` (list item) elements.


    13. `<ol>`:

      • Creates an ordered (numbered) list.

      • Contains one or more `<li>` elements.


    14. `<li>`:

      • Represents a list item within `<ul>` or `<ol>` lists.


    15. `<table>`:

      • Defines a table.

      • Contains one or more `<tr>` (table row) elements.


    16. `<tr>`:

      • Represents a table row within a `<table>`.

      • Contains one or more `<td>` or `<th>` elements.


    17. `<td>`:

      • Represents a table cell within a `<tr>`.

      • Contains the actual data in a table.


    18. `<th>`:

      • Represents a table header cell within a `<tr>`.

      • Typically used to label columns or rows in a table.


    19. `<form>`:

      • Creates a form to collect user input.

      • Can contain various form elements like `<input>`, `<textarea>`, and `<button>`.
